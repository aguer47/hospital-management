# Hospital Management API

## Project Description
The Hospital Management API is a RESTful API built using Node.js, Express, and MongoDB Atlas. The system helps manage hospital operations by storing and handling information about patients, doctors, appointments, and departments.

This project was developed as part of the CSE 341 Final Project.

---

## Features
- CRUD operations for hospital records
- MongoDB Atlas database integration
- REST API endpoints
- Swagger API documentation
- Error handling and validation
- OAuth authentication (planned)
- Deployment on Render

---

## Technologies Used
- Node.js
- Express.js
- MongoDB Atlas
- Swagger UI
- Render
- dotenv
- cors

---

## API Endpoints

### Patients
- GET /patients
- GET /patients/:id
- POST /patients
- PUT /patients/:id
- DELETE /patients/:id

### Doctors
- GET /doctors
- GET /doctors/:id
- POST /doctors
- PUT /doctors/:id
- DELETE /doctors/:id
